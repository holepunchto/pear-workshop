# Exercise 5 - Connecting Peers

This is a group exercise.

> **CAUTION**: DO NOT RUN UNTRUSTED PEAR LINKS

A peer who has the app installed can receive updates directly from the seeded drive. Share your Pear link with the group.

Create or join a topic in the chat app and start chatting across peers.

There are two layers of peer-to-peer happening here:

1. **Deployment** — the app itself is delivered peer-to-peer via the Pear Drive
2. **Application** — inside the app, peers connect and chat over Hyperswarm
